# Componentes principales

Aquí listamos las ubicaciones de los componentes y una breve descripción para que puedas encontrarlos rápidamente.

- `src/components/layout/Header.jsx` - Header global de la app.
- `src/components/layout/Footer.jsx` - Footer.
- `src/components/public/TripSearch.jsx` - Componente de búsqueda de viajes (origen, destino, fecha, filtros).
- `src/components/public/SeatMap.jsx` - Mapa de asientos público.
- `src/components/public/SeatSelection.jsx` - Selección de asientos para compra.
- `src/components/admin/*` - Paneles y herramientas administrativas.
- `src/components/ui/*` - Componentes UI reutilizables (`button`, `input`, `select`, `card`, etc.).
