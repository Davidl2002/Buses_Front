# Guía de contribución

¡Gracias por querer contribuir!

## 1. Cómo contribuir
1. Haz un fork del repositorio
2. Crea una rama nueva:
   git checkout -b feature/nombre-cambio
3. Aplica tus cambios
4. Haz commit y push
5. Envía un Pull Request a `develop`

## 2. Reglas del código
- Usa buenas prácticas
- Sigue la estructura del proyecto
- Asegúrate de que tu PR pase todas las revisiones

## 3. Comunicación
- Usa Issues para reportar bugs o proponer mejoras
- Usa Pull Requests para enviar código
